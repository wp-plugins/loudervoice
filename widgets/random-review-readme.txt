LouderVoice customers can use this plugin to add a random reviews widget to their sidebar(s).

== Description ==

The widget randomly shows one of your last 30 reviews. It shows the star rating, the author, an excerpt from the review and a link to where the full review can be read.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload and unzip the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance->Widgets and add the Random Reviews Widget where you want it.
4. Enter your LouderVoice Customer API Key
5. Enter how many characters of the review that you want to show
